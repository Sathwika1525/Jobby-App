# Jobby App – Job Portal

A responsive job portal built using React.js that allows users to browse, search, and filter job listings with secure authentication.

## 🚀 Features

* User authentication using JWT
* Protected routes for authorized access
* Job search functionality
* Filter jobs by employment type
* Filter jobs by salary range
* Detailed job description pages
* Loading, error, and empty state handling
* Fully responsive design for mobile and desktop

## 🛠 Tech Stack

* React.js
* JavaScript (ES6)
* CSS
* REST APIs
* JWT Authentication

## 📌 Description

Developed a responsive job portal using React.js with secure JWT-based authentication and protected routes. Managed loading, error, and empty states with reusable components and conditional rendering. Optimized the user interface for mobile and desktop to ensure a clean and consistent user experience.
